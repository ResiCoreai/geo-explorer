# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 05-01-2025

### Added
- Base map switch [#41](https://github.com/ncsa/geo-explorer/issues/41)
- Basic map controls [#42](https://github.com/ncsa/geo-explorer/issues/42)
- Basic point-and-click map interaction [#43](https://github.com/ncsa/geo-explorer/issues/43)
- Temporal data layer summary in sidebar [#44](https://github.com/ncsa/geo-explorer/issues/44)
- Collapse and expand button for the exploration page sidebar [#45](https://github.com/ncsa/geo-explorer/issues/45)
- Data Inventory advance controls [#46](https://github.com/ncsa/geo-explorer/issues/46)
- Data inventory component with automatic grouping by geometry type [#47](https://github.com/ncsa/geo-explorer/issues/47)
- Data table component for WFS feature attributes [#48](https://github.com/ncsa/geo-explorer/issues/48)
- Dataset summary to exploration page popup [#49](https://github.com/ncsa/geo-explorer/issues/49)
- Date time selector for temporal data [#50](https://github.com/ncsa/geo-explorer/issues/50)
- Decouple geo-explorer from DACHub [#6](https://github.com/ncsa/geo-explorer/issues/6)
- Export the components as default exports and register them in the component registry [#31](https://github.com/ncsa/geo-explorer/issues/31)
- Legend control [#51](https://github.com/ncsa/geo-explorer/issues/51)
- List temporal data in the data inventory panel [#52](https://github.com/ncsa/geo-explorer/issues/52)
- Map layers component with drag-and-drop reordering and visibility toggles [#53](https://github.com/ncsa/geo-explorer/issues/53)
- Pop-out draggable layer details/settings panel with data table and SLD style editor [#54](https://github.com/ncsa/geo-explorer/issues/54)
