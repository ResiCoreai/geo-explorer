# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 05-01-2025

### Added
- Base map switch [#166](https://github.com/ncsa/DACHub/issues/166)
- Basic map controls [#94](https://github.com/ncsa/DACHub/issues/94)
- Basic point-and-click map interaction [#232](https://github.com/ncsa/DACHUB/issues/232)
- Climate data layer summary in sidebar [#193](https://github.com/ncsa/DACHUB/issues/193)
- Collapse and expand button for the exploration page sidebar [#128](https://github.com/ncsa/DACHUB/issues/128)
- Data Inventory advance controls [#101](https://github.com/ncsa/DACHUB/issues/101)
- Data inventory component with automatic grouping by geometry type [#88](https://github.com/ncsa/DACHub/issues/88)
- Data table component for WFS feature attributes [#99](https://github.com/ncsa/DACHub/issues/99)
- Dataset summary to exploration page popup [#162](https://github.com/ncsa/DACHub/issues/162)
- Date time selector for climate data [#192](https://github.com/ncsa/DACHub/issues/192)
- Decouple geo-explorer from DACHub [#6](https://github.com/ncsa/geo-explorer/issues/6)
- Export the components as default exports and register them in the component registry [#31](https://github.com/ncsa/geo-explorer/issues/31)
- Legend control [#7](https://github.com/ncsa/DACHUB/issues/7)
- List climate data in the data inventory panel [#194](https://github.com/ncsa/DACHUB/issues/194)
- Map layers component with drag-and-drop reordering and visibility toggles [#89](https://github.com/ncsa/DACHUB/issues/89)
- Pop-out draggable layer details/settings panel with data table and SLD style editor [#93](https://github.com/ncsa/DACHUB/issues/93)
